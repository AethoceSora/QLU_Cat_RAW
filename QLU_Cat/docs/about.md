本项目使用MKDocs构建而成，遵循MIT开源协议。源代码已经开放至GitHub。<br/>

<br/>

如果大家有GitHub账户，请为本项目点Star吧！<br/>

 -基于Git共同编辑页面，实现网站的可持续发展<br/>
