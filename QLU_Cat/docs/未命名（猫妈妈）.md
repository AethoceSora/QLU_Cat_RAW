# 猫妈（黑黄）

**警告：这是一只带有幼崽的母猫，警惕性强，怕人，具有攻击性。曾抓伤路人。**

![猫妈妈](img/猫妈1.jpg)



常驻于操场北门，教师公寓栅栏附近。



识别特征：黑黄相间毛色，大概率带领一只黄色猫崽游荡。

![猫妈妈和小崽](img/猫妈&崽1.jpg)



曾经育有两只小猫崽，2021年11月20日左右，有一只小猫崽夭折或被人抱走。

![曾经的两只小猫崽](img/猫妈&两崽.jpg)
