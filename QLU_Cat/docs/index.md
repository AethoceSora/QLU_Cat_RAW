# 工大猫窝

如你所见，这是一个齐工大的**猫咪图鉴**网站。<br/>

灵感来自北京大学的同类项目。<br/>

目前本网站采用猫咪常出没的地点作为分类标准<br/>

希望各位撸猫愉快 ^ ^ <br/>
